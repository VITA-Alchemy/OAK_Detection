# OAK_nanodet-plus
OAK设备上适配nanodet-plus

# Environment
```bash
pytorch >= 1.7.0    # export.py  need
opencv-python >= 4.5.2.52
depthai   2.13.3.0
```

# RUN
```bash
1.首先转换nanodet-plus模型到onnx，官方自带转换工具（opset 12 ）。
2.使用在线转换工具 http://blobconverter.luxonis.com/ 
```
